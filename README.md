# hemi9536.github.io — Personal Portfolio

A multi-page digital portfolio hosted via GitHub Pages.

## Pages

| File | Page |
|------|------|
| `index.html` | Welcome / About Me |
| `resume.html` | Resume / CV |
| `interests.html` | Interests & Extracurriculars |
| `projects.html` | Projects & Academic Highlights |
| `social.html` | Social Media |

## How to customize

1. Replace every `[bracketed placeholder]` with your real content.
2. Add your photos to the `images/` folder and update the `src` attributes in the HTML files.
3. On `resume.html`, replace `YOUR_GOOGLE_DRIVE_FILE_ID` with your actual Drive file ID (or swap the iframe for a local PDF path).
4. On `social.html`, update each profile URL and write your 25-word brand description per platform.
5. Update `Your Name` in the `<nav>` brand and `<footer>` across all five pages.
